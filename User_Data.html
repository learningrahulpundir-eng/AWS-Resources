#!/bin/bash # Update system yum update -y # Install Apache yum install -y httpd # Start Apache systemctl start httpd systemctl enable httpd # Get IMDSv2 token TOKEN=$(curl -s -X PUT "http://169.254.169.254/latest/api/token" \ -H "X-aws-ec2-metadata-token-ttl-seconds: 21600") # Fetch instance metadata INSTANCE_ID=$(curl -s -H "X-aws-ec2-metadata-token: $TOKEN" \ http://169.254.169.254/latest/meta-data/instance-id) # Create HTML page cat < /var/www/html/index.html AWS EC2 Info

✅ EC2 Instance is Running

This page is created using User Data

Instance Details

Instance ID: $INSTANCE_ID

Hello from your Linux server 🚀

EOF
